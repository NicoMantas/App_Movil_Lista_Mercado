# Lista de Mercado App

Aplicación móvil nativa para Android desarrollada en Kotlin que permite gestionar listas de compras de manera sencilla y offline.

## Descripción

Esta aplicación permite a los usuarios crear y administrar listas de mercado de forma local en su dispositivo. Ideal para organizar las compras del hogar sin necesidad de conexión a internet.

## Tecnologías

- **Lenguaje:** Kotlin
- **IDE:** Android Studio
